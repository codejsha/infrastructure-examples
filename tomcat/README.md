# Apache Tomcat

## Install

### Binary/Archive

```bash
### pre-install
bash ./helper.sh

### install
bash ./install-tomcat.sh
```

### Chocolatey

```ps1
choco install -y tomcat

# choco uninstall -y tomcat
```
