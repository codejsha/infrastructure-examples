# PostgreSQL Source Connector

sample database: dvdrental (cf. [PostgreSQL README](/postgresql/README.md))

include/exclude configurations:

```json
    "database.dbname": "",
    "schema.include.list": "",
    "schema.exclude.list": "",
    "table.include.list": "",
    "table.exclude.list": "",
    "column.include.list": "",
    "column.exclude.list": "",
```
