# Infrastructure Examples

![Markdown Link Check](https://github.com/codejsha/infrastructure-examples/actions/workflows/link-check.yml/badge.svg)

## Overview

A collection of examples to help with development environment setup and infrastructure operations.

## Contents

- **Languages**: Kotlin, Java, Python, TypeScript, Go, C++, C#
- **Git**: Git, Git Hooks, GitHub, Gitea, GitKraken
- **Container**: Docker, Docker Compose, Podman, Harbor
- **Orchestration**: Kubernetes, K3s, kind, Minikube, Kubespray, Helm, ChartMuseum, Kustomize, Krew, etcd, Rook, Velero, CoreDNS, Istio, Kong, MetalLB
- **Cloud & IaC**: Terraform, Ansible, AWS, LocalStack
- **CI/CD**: Tekton, Jenkins, Argo CD, Nexus
- **Security**: OpenSSL, GnuPG, HashiCorp Vault, Keycloak, cert-manager, OpenLDAP
- **OpenAPI**: OpenAPI, OpenAPI Generator, Postman, Swagger, Redoc, Scalar, Stoplight Elements, oasdiff, Prism, Schemathesis, AsyncAPI
- **Database**: MySQL, PostgreSQL, SQL Server, Oracle Database, Redis, MongoDB, Flyway, Jooq
- **Big Data**: SeaweedFS, MinIO, Hadoop, Zookeeper, Elasticsearch
- **Messaging**: Kafka, Confluent, Kafka Connect, Flink, ksqlDB, Kafka UI, Kafka Lag Exporter, Kafdrop, kcat, Burrow, CMAK
- **Observability**: Prometheus, Grafana, Alertmanager
- **Web Server**: Apache HTTP Server, Apache Tomcat, Nginx, IIS, WebLogic Server, WildFly
- **Linux**: Linux, Make, Neovim, KVM, FTP, Starship, Atuin, LSD, Tabby, Alacritty, Hyper
- **Windows**: Windows, PowerShell(pwsh), Hyper-V
- **macOS**: macOS, zsh, Karabiner-Elements, Rectangle, hostctl
